# Project Report

## Title
Strategies With Tableau: A College Food Choices Case Study

## Introduction
This project analyzes the food choices of college students using Tableau. The aim is to understand eating habits and identify factors that influence students' food preferences.

## Objectives
- Analyze college food choice data.
- Create interactive dashboards using Tableau.
- Visualize trends and patterns.
- Generate meaningful insights for better decision-making.

## Dataset
The dataset contains information about students' food habits, preferences, health awareness, and lifestyle factors.

## Methodology
1. Collected the dataset.
2. Cleaned and prepared the data.
3. Imported the dataset into Tableau.
4. Created charts and dashboards.
5. Analyzed the results.

## Tools Used
- Tableau
- GitHub
- CSV Dataset

## Results
The dashboard helps identify common food preferences, healthy eating patterns, and factors affecting students' food choices.

## Conclusion
Tableau makes it easy to visualize data and understand student food habits through interactive dashboards.
