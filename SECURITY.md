# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.0   | ✅ Yes |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it to the project maintainer.

Please include:
- A description of the issue
- Steps to reproduce the problem
- Expected behavior
- Screenshots (if applicable)

We will review the report and address the issue as soon as possible.

Thank you for helping improve the security of this project.
