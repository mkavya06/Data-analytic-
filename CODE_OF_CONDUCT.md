# Code of Conduct

## Our Pledge

We are committed to making participation in this project a respectful and welcoming experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful and supportive.
- Accepting constructive feedback.
- Using inclusive and professional language.
- Helping others learn and grow.

Examples of unacceptable behavior include:

- Harassment or discrimination.
- Offensive language or personal attacks.
- Sharing private information without permission.

## Enforcement

Project maintainers have the right to remove or edit comments, commits, or contributions that violate this Code of Conduct.

## Contact

If you have any questions regarding this Code of Conduct, please contact the project maintainer.
